## 指令

| 指令        | 描述                        |
| ----------- | --------------------------- |
| help        | 显示帮助文档                |
| agent       | 运行agent程序               |
| avro-client | 运行avro-client客户端       |
| password    | 创建用于flume配置的密码文件 |
| version     |显示Flume的版本信息|
